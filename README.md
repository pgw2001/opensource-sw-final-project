## 2. 프로젝트 개요 📋
- 미니 게임들을 모아서 플레이할 수 있도록 **파이썬 & Streamlit**을 사용.
- **파이썬 언어로 개발**을 진행하며, 플레이어와 상호작용 시스템 구축을 위해 **프론트 엔드는 Streamlit**을 사용.
- 랭킹, 업적 기능을 추가하여 플레이어가 게임 플레이에 대한 흥미를 끌어올리도록 구현.
- 플레이 가능한 게임은 실시간으로 계속 추가 예정(2048, 스도쿠 등).
